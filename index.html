PROJECTS

Spinner Loading Bar

This was a small but meaningful project where I built a loading animation using only HTML and CSS. The goal was to create a simple, smooth spinner that could be used as a loading indicator on websites or web apps.
Key features:
Fully created with HTML and CSS (no JavaScript)
Used @keyframes for smooth rotation
Practiced using transform, border, and animation timing
What I learned:
This project improved my understanding of CSS animations, timing functions, and visual optimization for UI elements.

FC Barcelona Fan Page Website

This project was inspired by my passion for football and especially FC Barcelona. I designed and coded a full fan page website that includes multiple sections to reflect the spirit of the club.
Key sections of the website:
Homepage with the club's logo and welcome message,
Club history and achievements,
A list of legendary players (with images),
Match highlights and upcoming fixtures,
Photo gallery section.
Technologies used: HTML, CSS
What I learned:
I improved my ability to organize content in a clean structure, apply responsive styling, and match visual themes to a brand. It also helped me build a full multi-section static website from scratch.

Landscape

In this project, I designed a clean and visually appealing sunny landscape using only HTML and CSS. The goal was to create a simple yet effective vector-style illustration featuring key visual elements such as the sun, clouds, grass, and sky—all styled purely with CSS.
Technologies used:
HTML5 – for structure
CSS3 – for visual styling and layout
This project showcases my ability to use CSS creatively to design custom shapes and build engaging visual compositions without relying on images or JavaScript.
